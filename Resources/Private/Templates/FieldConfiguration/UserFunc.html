Important Information

If you want to use this field, please note that frontend and backend parameters of your userFunc are different!
So, please only use the 'parameters' key in the $config Array from the first argument of your called userFunc.

See: 

namespace VendorName\ExtensionName\UserFunc;

class YourUserFunc
{  
    /**
     * The custom userFunc Method that only consists
     * of $config["parameters"] as set up in the
     * field configuration
     *
     * @param array $config Configuration and Parameters Array
     * @param mixed $funcRef Parent Object Reference
     * @return mixed
     */
    public function userFuncMethod($config, &$funcRef)
    {
        // The following parameters are the same between backend
        // and frontend and always deliver the parameters, that
        // you've set up in the field configuration
        $parameters = $config["parameters"];
    }
}
