DataViewer Information Template
You are seeing this, because you didn't select a template in the plugin settings.
Templates can either be manually selected by a file or they can be added in TypoScript like this: 

plugin.tx_dataviewer.templates {
    General Templates = --div--
    MyFirstTemplate = fileadmin/templates/dataviewer/Templates/MyFirstTemplate.html
    ASecondTemplate = EXT:custom/Resources/Private/Templates/Dataviewer/ASecondTemplate.html
}
{_all}
